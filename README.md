 **QFComponent for lazarus**   
秋风(QQ:315795176)开发的控件包，有2个控件：  
TQFRichView：类RichView控件  
TQFScrollingText：滚动显示控件  
使用以下特定符号定义文字（行）的属性：  
[img]本地图像文件名称  
[!]下划线  
[@]删除线  
[#]粗体  
[$]斜体  
[L]行居左  
[C]行居中  
[R]行居右  
[C1]黑色  
[C2]红色  
[C3]黄色  
[C4]绿色  
[C5]蓝色  
[S1]字体尺寸9  
[S2]字体尺寸12  
[S3]字体尺寸14  
[S4]字体尺寸16  
[S5]字体尺寸18  
[LINE]分割线  
[2LINE]双线条分割线  
定义文字颜色、样式：  
<C1>xxx</C>  
<C1>黑色  
<C2>红色  
<C3>黄色  
<C4>绿色  
<C5>蓝色 
</C>恢复原来的文字颜色  
<!>下划线  
<@>删除线  
<#>加粗  
</>恢复原来的文字样式
初步 **支持markdown格式的表格**   
图像格式支持： **jpg,bmp,png** 等    
可在windows和linux使用，已在龙芯电脑实测可用。  
TQFRichView控件支持鼠标中键滚动及按鼠标左键然后上下移动鼠标实现快速滚动。
